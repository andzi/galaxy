About This Website

For the purpose of learning more about non-Flash web animation, I made this website as a simple idea of what can be done without Flash. This website utilizes jQuery, some jQuery plugins and my custom jQuery code to give you this visual experience.

This website is aimed at people some-what familiar with HTML, CSS and JavaScript. Although you won't find any code displayed on the page, please look at the source if you are more interested in how things work.

This website is fully compliant with web standards (HTML & CSS).

How it Works

This website leans on JavaScript to produce that animations. More specifically, this website uses the jQuery JavaScript library. jQuery describes itself as, "The Write Less, Do More, JavaScript Library," and it lives up to that description.

See for yourself by looking at the main JavaScript file.
